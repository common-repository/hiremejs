If u want to tell somebody something in his Developer Javascript Console use this Plugin.
You can use a Figlet as Header and free text.

== Description ==

Sometimes you want to hire some person for your Company. Most of the Developer have a look at the source code of your company when they visit you site.
So show them some Information and tell them you are a great and hilarious company ;)

If you have Questions or Feature Request, feel free and contact me.

== Installation ==

1. Upload `HireMeJs.zip` to the `/wp-content/plugins/` directory
2. Extract `HireMeJs.zip`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Wordpress Settings -> HireMeJs (SubMenu of WP main settings)
5. Add the information you want to show the developer world
6. you have to use `<?php wp_footer(); ?>` in your Theme so this Plugins works
